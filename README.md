# Project description

A Pokedex app that uses the PokeAPI to make a Kanto league Pokedex. Made using HTML, CSS and vanilla Javascript.

# ⚒️ Built with

<ul>
  <li>HTML 5</li>
  <li>CSS 3</li>
  <li>Javascript</li>
</ul>

# How to install

Clone the repository, navigate to your preferred code editor and navigate to index.html.
